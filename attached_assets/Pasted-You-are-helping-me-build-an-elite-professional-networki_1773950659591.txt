CONNECTION / INTRODUCTION FLOW

Step 1 — Match surfaces
  - Monthly subscribers receive curated weekly recommendations
  - Free/credit users browse and manually find people

Step 2 — Expression of interest
  - User A clicks "Request Connection" on a profile
  - No credit is charged yet
  - Request enters pending state (7-day window)

Step 3 — Mutual confirmation
  - User B is notified of the connection request
  - User B accepts or declines
  - If User B accepts → 1 credit is charged to User A at this moment
  - If User B declines → no charge, request closes

Step 4 — Introduction made
  - Both users can now message each other
  - Meeting scheduling becomes available

Credit edge cases:
  - If User A has 0 credits when User B accepts, show message:
    "Your connection request to [Name] is pending. Add credits to complete 
    this connection." Hold the request open for 7 days from acceptance.
  - If User A does not add credits within 7 days of User B accepting, 
    the request expires and User B is notified it is no longer available.
  - Credits are NEVER charged at meeting booking — only at mutual acceptance.
